# Zotero Translation Server

The Zotero translation server lets you use [Zotero translators](https://www.zotero.org/support/translators) without the Zotero client.

This repo is a fork from the original [zotero/translation-server] but has been modified to support deployment on Vercel.

## Installation

### Vercel

Fork this repository to your own GitHub account then deploy.

> [!Warning]
> Text Search is not supported on Vercel for now.
